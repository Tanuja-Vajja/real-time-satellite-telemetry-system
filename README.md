# Real-Time Satellite Telemetry System 🚀

A real-time telemetry data visualization and decoding system developed using Python and PyQt5 for satellite communication simulation.

## 📌 Project Overview

This project simulates the reception, decoding, and visualization of telemetry data using a client-server architecture. The system processes telemetry frames and displays decoded parameters through a graphical user interface.

The project demonstrates concepts related to:

* Satellite Telemetry
* CCSDS Communication Concepts
* Socket Programming
* Real-Time Data Processing
* GUI Development using PyQt5 and Tkinter

## ✨ Features

* Real-time telemetry simulation
* Client-server communication

* GUI-based telemetry visualization
* Telemetry log generation
* Simulated satellite parameter monitoring

---

## 📂 Project Structure

* `client.py` → Client-side telemetry receiver
* `server.py` → Server-side telemetry sender
* `gui_tkinter.py` → Tkinter GUI implementation
* `sample_simulated_data.py` → Sample telemetry data generator
* `simulated_for_pyqt.py` → PyQt5 simulation module
